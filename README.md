# VETCENTER CLINIC 

Веб-сайт ветеринарной клиники "VETCENTER CLINIC" с информацией о услугах, специалистах, контактах и онлайн-записью для питомцев.

## Технологии

- HTML5
- CSS3
- JavaScript (для модального окна онлайн-записи)
- Адаптивная верстка для мобильных и десктоп устройств

## Структура проекта

- `index.html` — основной HTML файл сайта
- Встроенные стили CSS и скрипт для работы модального окна

  ## Запуск локально

1. Скачайте проект на свой компьютер.
2. Откройте файл `index.html` в браузере.
3. Наслаждайтесь функциональным и адаптивным сайтом.
